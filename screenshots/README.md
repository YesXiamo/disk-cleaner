# 截图目录

此目录用于存放应用截图，供 README 和分享文档使用。

## 建议的截图

1. **app-screenshot.png** - 主界面截图（必需）
   - 建议尺寸：1200x800 或更高
   - 显示热力图和侧边栏

2. **scan-result.png** - 扫描结果截图
   - 展示树状图效果

3. **context-menu.png** - 右键菜单截图
   - 展示右键功能

4. **file-details.png** - 文件详情弹窗截图

## 截图提示

- 使用 Mac 自带截图工具：`Cmd + Shift + 4`
- 或使用 `Cmd + Shift + 5` 选择窗口截图
- 建议清理敏感信息后再提交

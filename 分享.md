# 因为电脑磁盘满了，所以我写了个磁盘清理工具 🤦‍♂️

> 一个被"您的磁盘空间不足"逼出来的开源项目

## 故事的开始

事情是这样的...

某天我正在愉快地码代码，突然弹出一个熟悉的提示框：

**「您的磁盘空间不足，仅剩 2GB 可用」**

我：？？？

我明明记得上周还有 50GB 的！这 Mac 的存储都去哪儿了？？

于是我开始了一场找文件之旅：
- 打开 Finder → 进入各个文件夹 → 查看简介 → 记录大小...
- 半小时后，我累了 😫
- 而且根本看不出来哪个文件夹占用了最多空间！

作为一个程序员，我怎么能忍受这种低效的操作？

**于是，我决定自己写一个！**

## Disk Cleaner 诞生了

### 核心功能

🔥 **树状图热力图** - 一眼就能看出谁是大胃王
- 方块越大 = 占用越多
- 颜色越红 = 文件越大
- 再也不用猜哪个文件夹在偷吃你的硬盘了

🖱️ **右键菜单** - 操作快如闪电
- 打开 / 在 Finder 中查看 / 删除
- 三件套齐全，想干嘛就干嘛

👻 **隐藏文件也无所遁形**
- 那些以 `.` 开头的文件夹
- 平时看不见的缓存、配置，全给你揪出来

🗑️ **删除后手动刷新**
- 删完不用等扫描，想什么时候刷新都行
- 急性子福音

### 技术栈

- **Electron** - 跨平台桌面应用框架
- **Node.js** - 后端文件系统操作
- **HTML/CSS/JavaScript** - 前端界面
- **Flex 布局** - 树状图自适应布局

## 安装使用

### 开发者模式

```bash
# 克隆代码
cd /Users/dww-cainiao/AI/testiflow/disk-cleaner

# 安装依赖
npm install

# 运行
npm start
```

### 直接使用打包版

已经打包好的应用位于：
```
dist/mac-arm64/Disk Cleaner.app
```

双击即可运行，或者安装到 Applications：
```bash
cp -R "dist/mac-arm64/Disk Cleaner.app" /Applications/
```

## 使用技巧

### 1. 快速定位大文件
- 看颜色：越红的越大
- 看大小：方块面积 = 占用比例
- 右键 → "在 Finder 中打开" → 直接定位

### 2. 清理策略
- 先扫根目录，看哪个用户文件夹最大
- 进入该文件夹，继续深挖
- 找到罪魁祸首（通常是 node_modules、缓存、日志等）

### 3. 隐藏文件别乱删
- `.` 开头的文件谨慎删除
- 不确定的先用 "查看详情" 确认
